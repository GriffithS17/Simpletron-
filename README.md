# Simpletron-
Fazendo um computador sem precisar soldar algum componente. Vi esse exercício no livro C: como programar, de Paul Deitei e Harvey Deitel. O programa simula um computador capaz de ler, escrever, somar, diminuir, multiplicar, dividir, mudar a posição dos valores na memória. Fique a vontade para modificar e melhorar.
